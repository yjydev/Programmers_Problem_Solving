## 체육복

```python    
def solution(n, lost, reserve):
    r_lost = set(lost) - set(reserve)
    r_reserve = set(reserve) - set(lost)
    answer = n-len(r_lost)
    for r in r_reserve:
        if r-1 in r_lost:
            answer += 1
            r_lost.remove(r-1)
        elif r+1 in r_lost:
            answer += 1
            r_lost.remove(r+1)
    return answer
```
