## 나머지가 1이 되는수 찾기    
- 풀이   

```python   
def solution(n):
    for i in range(1,n):
        if n % i == 1:
            return i
```      
- 자연수 n이 주어지고, n을 x로 나눴을때 나머지가 1이 되는 가장 작은 x를 찾는 문제였다.   
- 풀이가 너무 짧아서 사실 올릴지 말지 되게 고민되는 코드였다.   
- 다른 문제들에 비해 푼 사람 수가 적었던 것 같았는데, 어려워서가 아니라 너무 간단해서였나보다.    
- 사실 너무 간단하다보니 그냥 순회를 돌면 시간초과가 되거나 등의 어려움을 예상했는데 바로 통과가 되서 조금 놀랐다.
