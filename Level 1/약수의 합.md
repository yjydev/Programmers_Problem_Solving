## 약수의 합   
- 풀이   

```python   
def solution(n):
    answer = 0
    for k in range(1,n+1):
        if n % k == 0:
            answer += k
    return answer
```     
- 이번 문제는 주어진 수의 약수들의 합을 구하는 문제였다.    
- 간단하게 생각하면, 주어진 수가 n에 대하여, n-1 까지의 모든 수들 중, 나누어 떨어지는 수들의 합을   
  구하면 되는 문제였다.    
- 다른 풀이를 살펴보니, `n + (n//2 까지의 약수의 합)` 으로 구하면 성능이 훨씬 더 좋다는 것을 알 수 있었다.
